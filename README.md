# 旅游微信小程序练习
## 说明：
关于一个旅游景点的小程序练习，很适合初级者来学习，新的功能和界面还在不断更新中......
## 目录结构：
* images-存放项目图片
* pages-存放独立的文件
## 功能
实现百度地图，分享，日期选择器，电话拨打,获取用户权限一些功能......
